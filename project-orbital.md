STK (Systems Tool Kit) is a professional aerospace simulation and visualization software developed by AGI/Ansys. It allows engineers and mission designers to model satellites, spacecraft, aircraft, sensors, ground stations, orbits, coverage, and mission performance—all in a realistic 3D environment.

### Core Competencies & Applications
STK helps to:
1. Visualize space missions
2. Design and analyze orbits
3. Compute coverage and visibility
4. Mission analysis & operations planning
5. Multi-domain simulation

STK can model Space + Ground + Air + Land + Sea.

### Project Highlights
* Built scenarios and orbits using classical orbital elements
* Visualized ground tracks across orbit types (LEO, Molniya, GEO, sun-synchronous)
* Calculated access and visibility windows between assets
* Explored J2 perturbation effects on orbital dynamics

**Certifications:** STK Level 1 Certification
