Store all site images here.

Suggested layout:

- `home/` for homepage visuals used in `index.html`
- `about/` for profile or timeline images used in `about.html`
- `publications/` for publication covers, charts, or thumbnails
- `shared/` for logos, icons, or visuals reused across multiple pages
- `projects/<project-name>/` for images specific to each project page

Naming tips:

- Use short lowercase file names with hyphens, for example `ngc4852-diagram.jpg`
- Use `.jpg` for photos, `.png` for transparent images, and `.svg` for vector graphics when possible
- Keep filenames descriptive so they are easy to match in HTML
